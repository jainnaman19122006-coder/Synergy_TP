# Linux Commands Documentation

## 1. pwd
Command:
pwd

Purpose:
Displays the current working directory.

Example:
pwd

---

## 2. ls
Command:
ls

Purpose:
Lists files and directories in the current location.

Example:
ls

---

## 3. ls -la
Command:
ls -la

Purpose:
Lists all files including hidden files with detailed information.

Example:
ls -la

---

## 4. cd
Command:
cd

Purpose:
Changes the current directory.

Example:
cd task_1

---

## 5. mkdir
Command:
mkdir

Purpose:
Creates a new directory.

Example:
mkdir test_folder

---

## 6. touch
Command:
touch

Purpose:
Creates an empty file.

Example:
touch sample.txt

---

## 7. cat
Command:
cat

Purpose:
Displays the contents of a file.

Example:
cat sample.txt

---

## 8. echo
Command:
echo

Purpose:
Prints text to the terminal or writes text into a file.

Example:
echo Hello World

---

## 9. cp
Command:
cp

Purpose:
Copies files or directories.

Example:
cp file1.txt file2.txt

---

## 10. mv
Command:
mv

Purpose:
Moves or renames files and directories.

Example:
mv old.txt new.txt

---

## 11. rm
Command:
rm

Purpose:
Deletes files or directories.

Example:
rm file.txt

---

## 12. grep
Command:
grep

Purpose:
Searches for a pattern in a file.

Example:
grep "hello" sample.txt

---

## 13. find
Command:
find

Purpose:
Searches for files and directories.

Example:
find . -name "*.py"

---

## 14. head
Command:
head

Purpose:
Displays the first 10 lines of a file.

Example:
head file.txt

---

## 15. tail
Command:
tail

Purpose:
Displays the last 10 lines of a file.

Example:
tail file.txt

---

## 16. wc
Command:
wc

Purpose:
Counts lines, words, and characters in a file.

Example:
wc file.txt

---

## 17. chmod
Command:
chmod

Purpose:
Changes file permissions.

Example:
chmod 755 script.sh
